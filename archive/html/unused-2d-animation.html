Pilotage in São Luís

An unfinished 2D animation for a pilotage company, focused on providing an overview of the current pilotage process to a lay audience.

Pilotage in São Luís is an unfinished 2D animation for for SmartPilotsMa, focused on providing an overview of the current pilotage process to a lay audience.
Timeframe
1 week
Tools
Adobe Illustrator, Adobe After Effects
PROBLEM
The client identified inconsistencies in pilots’ scheduling but needed to communicate these issues to a non-specialist audience (e.g., stakeholders or public groups). Given the technical complexity of pilotage regulations (e.g., duty cycles, rest periods, and jurisdictional rules), a simplified yet accurate visual explanation was required to gather actionable feedback.
